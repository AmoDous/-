# Rooms

Rooms — рабочий прототип адаптивного сайта для поиска и бронирования приватных помещений: лаунджей, детских комнат, караоке-комнат, банкетных залов, переговорных и VIP-комнат.

Стартовый город MVP: Воронеж.

## Что сейчас есть

- Главная страница с подбором помещения по дате, времени, длительности и количеству гостей.
- Каталог демо-помещений с фильтрами по типу и удобствам.
- Логика 18+: кальян и алкоголь скрываются, если среди гостей есть несовершеннолетние.
- Альтернативные окна, если выбранное время занято.
- Карточка помещения с описанием, правилами, отзывами и параметрами.
- Демо-заявка на бронь с расчётом стоимости и предоплаты 30%.
- Локальная история заявок как прототип личного кабинета.
- Форма заявки для партнёров на подключение помещения.

## Как открыть

Основная версия GitHub Pages:

https://amodous.github.io/Rooms-bron/

Локально можно открыть `index.html` прямо в браузере.

## Важно

Это frontend-прототип. Для настоящего MVP нужны backend, платежи, кабинет партнёра, админка Rooms, модерация площадок и реальные данные помещений.

Старые файлы `mobile.html`, `web.html`, `app/*` и standalone-макет пока оставлены как архив визуальных вариантов, чтобы не потерять дизайн-материалы.
